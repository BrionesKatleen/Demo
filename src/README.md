# GUIDELINES (STRICT) 
## Project Structure
Stretch
-	/assets       → images, icons, UI files
-	/docs         → documentation
-	/src
  o	main.cs → main program,
  o	/models → data classes (e.g., TaskModel.cs)
  o	/repositories → database operations (e.g., TaskRepository.cs)
  o	/controllers → handle app logic (e.g., TaskController.cs)
  o	/services → reusable services/methods (e.g., TaskService.cs)
  o	/views → UI elements (forms, dashboards)
  o	README.md → instructions for this part
-	/database
  o	DBConnection.cs → database connection
•	README.md → project overview
